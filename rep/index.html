Bienvenue !

Cette plateforme vous permet d’explorer toutes les applications web générées automatiquement par les LLM dans le cadre du projet PJI-LLM.
Sélectionnez une version, puis un LLM pour découvrir et utiliser chaque application.
Structure du projet : PJI-LLM/LLM/vX/index.html

  • Chaque version correspond à une étape du projet
  • Les LLM testés sont : ChatGPT, Gemini et Perplexity
